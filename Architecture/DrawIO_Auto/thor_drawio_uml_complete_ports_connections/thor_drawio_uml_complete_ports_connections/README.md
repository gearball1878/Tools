# THOR UML Complete Ports and Connections

This version renders:

- UML-like block objects
- explicit visible port objects on the block boundaries
- connections terminating on ports
- top-down pages
- readable proprietary attributes via draw.io `Edit Data`

## Open

Open:

`thor_uml_complete_ports_connections.drawio`

in draw.io / diagrams.net.

## Inspect metadata

Select a block, port, or connection:

Right click → Edit Data

You will see fields such as:

- `kind`
- `object_id`
- `object_name`
- `stereotype`
- `level`
- `type`
- `source`
- `target`
- `parent_block_id`
- `fqid`
- `attr_*`
- `attributes_json`

## Run

```bash
python example_uml_complete.py
```
